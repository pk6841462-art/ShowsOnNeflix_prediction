* Title
Netflix Availability Prediction 
* Project Overview
This project uses Logistic Regression to predict whether a movie or TV show is available on Netflix based on its features.
* The goal is to build a classification model that predicts Netflix availability (Yes/No) using:
- Year
- Minimum Age (Age Restriction)
- IMDB Rating
- * Required tools
    Python
    Numpy
    Pandas
    Matplotlib
